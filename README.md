# Project structure for small projects

- functions.scss
- mixins.scss
- variables.scss

### base
- animations.scss
- base.scss
- typography.scss
- utilities.scss

### components
- component.scss

### layout
- navigation.scss
- objects.scss
- sections.scss

### pages
- homepage.scss

### themes
- default.scss

### vendors
Normalize, Bootstrap files, 3rd party libs

### OTHERS
- fixes.scss
